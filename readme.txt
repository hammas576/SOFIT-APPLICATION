This projects utilizes the {port:3000}
and contains apis for the following functionalities:

(1) api/employee
This API gets employee data in the form of a csv file extracts data from the file and inserts it in the database after validation

(2)api/getEmployeeByNumber
Gets a string from the user containing a phone number with first, middle or last digits and finds the user based on those

(3)api/addEmployeePicture
Adds a profile picture to an existing employee

(4)api/getEmployeeByEmail
Finds the employee with the entered email

(5)api/sendEmail
Sends an email to the entered user
